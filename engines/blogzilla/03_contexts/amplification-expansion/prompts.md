# Amplification & Expansion Prompts

## Thinking Prompt

“Review all generated outputs and identify themes that resonated strongly. Suggest five new blog topics, three collaboration or backlink opportunities, one lead magnet idea and one automation or process improvement. Explain briefly why each suggestion could amplify the signal.”

## File Editing Prompt

“Create the following files in the run folder: `new-topics.md`, `collaboration-opps.md`, `backlink-opps.md`, `lead-magnet.md`, `automation-improvement.md`. Populate each with the respective suggestions generated from the thinking prompt.”

## Folder Processing Prompt

“After saving the suggestions, create an index file (e.g. `expansion-summary-[topic]-v1.md`) listing all expansion outputs and a short status summary indicating readiness for the next cycle.”
