# Source Intake Context

## Purpose

Initiate a Blogzilla run by capturing the source material and establishing the run parameters. This context handles registration of the blog post, keyword or idea, selection of resonance mode and core intent, and creation of the run brief and run folder.

## Read Before Starting

- The source file or URL placed in `07_inputs/source-blogs/` or the keyword/idea in `07_inputs/keywords/`.
- The run brief template in `06_templates/run-brief-template.md` (if applicable).
- The system overview and workflow map in `02_system-definition/overview.md` and `02_system-definition/workflow-map.md`.

## Ignore

- Completed outputs in `08_outputs/`.
- Archive contents in `09_archive/`.

## Expected Outputs

- A completed run brief describing the topic, resonance mode, core intent, niche and any special constraints.
- A run folder created in `04_workspaces/source-intake/` with a slug based on the date and topic.
- Updated source registration notes.
