# Source Intake Prompts

## Thinking Prompt

“Given the input source, identify the main topic, niche, resonance mode (Authority, Emotional, Technical, Viral, Educational) and core intent (SEO, leads, thought leadership, audience growth). Produce a concise summary of the source in two sentences.”

## File Editing Prompt

“Create a run brief using the source material, resonance mode, core intent and niche. Include: a one‑sentence topic description, the chosen resonance mode, core intent, niche/domain, and any notes or constraints. Save as `run-brief-[topic]-v1.md` in the run folder.”

## Folder Processing Prompt

“Create a run folder named `run‑YYYY‑MM‑DD‑[topic-slug]/` under `04_workspaces/source-intake/` and place the run brief and any registered source files inside it.”
