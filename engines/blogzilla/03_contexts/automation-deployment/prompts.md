# Automation & Deployment Prompts

## Thinking Prompt

“Plan the publishing and scheduling for the current run’s outputs. Determine the optimal order and timing for each platform based on the recommended cadence and platform best practices. Identify which outputs require design/VA support and which can be automated via scheduling tools.”

## File Editing Prompt

“Create a deployment sheet `deployment-[topic]-YYYY-MM-DD.md` summarising platform, post filename, scheduled publish time, CTA tier and whether it is automated or manual. Draft any VA/Canva briefs as separate files (e.g. `design-brief-[topic]-v1.md`).”

## Folder Processing Prompt

“Group outputs by platform in the run workspace. Copy final versions into a staging folder for publishing. Update the platform outputs tracker with scheduling details and mark each item as queued or pending.”
