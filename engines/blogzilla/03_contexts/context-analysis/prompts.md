# Context Analysis Prompts

## Thinking Prompt

“Analyse the source and run brief to determine the tone, audience intent, emotional frequency and conversion goal. Identify the primary keyword and 3–5 LSI keywords. Suggest two long‑tail variants and three related questions. Recommend a publication cadence and silo. Compose a canonical statement linking back to the original blog.”

## File Editing Prompt

“Create an analysis file named `context-analysis-[topic]-v1.md` in the run folder. Include: tone description, audience intent summary, conversion goal classification, primary keyword, LSI keywords, long‑tail variants, related questions, publication suggestions and the canonical statement.”

## Folder Processing Prompt

“If multiple source files exist, normalise them into a single canonical analysis file by reading all documents in the run folder and synthesising a unified analysis.”
