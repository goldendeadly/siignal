# Platform Generation Prompts

## Thinking Prompt

“Using the summary, meta description, highlights and analysis data, generate tailored outputs for each selected platform (X/Twitter, LinkedIn, Instagram, YouTube, Quora, Facebook, Pinterest, Medium/Substack/Blogger/Dev.to). Ensure tone matches the chosen resonance mode, include primary and LSI keywords naturally, and embed appropriate CTAs and canonical links.”

## File Editing Prompt

“Create draft files for each platform in the run folder. For example: `thread-x-[topic]-draft-v1.md`, `carousel-instagram-[topic]-draft-v1.md`, `linkedin-article-[topic]-draft-v1.md`, `youtube-outline-[topic]-draft-v1.md`. Populate each file with the platform‑specific content following the workflow rules.”

## Folder Processing Prompt

“After generating drafts, organise them into subfolders by platform within the run folder if necessary, and produce a brief status summary listing all created files and their readiness level.”
