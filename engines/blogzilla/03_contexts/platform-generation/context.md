# Platform Generation Context

## Purpose

Convert the core assets into platform‑specific outputs. This context handles the creation of posts, threads, carousels, scripts and articles tailored to each platform’s format, tone and CTA structure while maintaining resonance mode, SEO layer and canonical links.

## Read Before Starting

- Summary, meta description and highlights from `core-generation`.
- Tone and keyword information from `context-analysis`.
- Platform guidelines from `01_source-material/platform-guides/` (if available).

## Ignore

- Outputs from unrelated platforms or previous runs.
- Amplification/expansion ideas until Phase 5.

## Expected Outputs

For each active platform (choose based on run brief):

- Draft posts or scripts adhering to the platform’s character limits and style.
- Appropriate CTA tiers (Soft, Engagement, Conversion) and hashtag sets.
- Canonical/backlink line referencing the original blog.
- Notes on suggested visuals (for carousels or videos) where relevant.
