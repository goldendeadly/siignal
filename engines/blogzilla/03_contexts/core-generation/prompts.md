# Core Generation Prompts

## Thinking Prompt

“Using the analysis results and the source content, create a concise summary (2–3 sentences) and a 150–160 character meta description. Identify 5–10 key insights and tag each as a Framework, Principle, Statistic, Provocation or Meta‑Insight.”

## File Editing Prompt

“Create the following files in the run folder: `summary-[topic]-v1.md`, `meta-description-[topic]-v1.md` and `key-highlights-[topic]-v1.md`. Populate them with the summary, meta description and tagged highlights respectively.”

## Folder Processing Prompt

“Batch‑format the highlights into a reusable table or list that can be consumed by downstream contexts. Ensure each highlight has an associated tag.”
