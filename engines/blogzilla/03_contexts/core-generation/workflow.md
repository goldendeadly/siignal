# Core Generation Workflow

1. **Read the analysis output** – familiarise yourself with the tone, keywords and audience insights.
2. **Draft summary** – write a two‑to‑three sentence summary capturing the core idea of the source. Keep it concise and engaging.
3. **Compose meta description** – craft a 150–160 character description summarising the article for search and social previews. Include the primary keyword.
4. **Extract highlights** – identify 5–10 key points, statistics or quotes from the source. Reformat each as a standalone insight.
5. **Tag highlights** – classify each highlight as a Framework, Principle, Statistic, Provocation or Meta‑Insight.
6. **Save assets** – save the summary (`summary-[topic]-v1.md`), meta description (`meta-description-[topic]-v1.md`) and highlights (`key-highlights-[topic]-v1.md`) into the run folder.
7. **Hand off** – move the run to `platform-generation` for output creation.
