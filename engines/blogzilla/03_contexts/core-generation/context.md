# Core Generation Context

## Purpose

Transform the analysed source into modular core assets: a concise summary, an SEO meta description and a list of key highlights tagged by type. These core assets serve as the seed for all downstream platform outputs.

## Read Before Starting

- The analysis file from `context-analysis` (e.g. `context-analysis-[topic]-v1.md`).
- The original source content.

## Ignore

- Platform‑specific drafts from later contexts.
- Archived runs and completed outputs.

## Expected Outputs

- A two‑to‑three sentence summary capturing the essence of the source.
- A 150–160 character meta description optimised for search and social previews.
- Five to ten key highlights, each rephrased as a standalone insight and tagged as one of: Framework, Principle, Statistic, Provocation or Meta‑Insight.
