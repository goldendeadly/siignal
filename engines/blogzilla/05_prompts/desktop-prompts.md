# Desktop Prompts

Desktop prompts are used for higher‑level thinking tasks—deciding the structure of a run, choosing resonance mode and intent, and planning next steps. They are executed in the ChatGPT UI or a similar interface where you provide instructions and interpret results.

Example prompt:

“You are operating in desktop mode. Read the run brief (`07_inputs/run-briefs/[run-brief].md`) and the source file (`07_inputs/source-blogs/[source-file]`). Determine the best Blogzilla run plan: choose the resonance mode, identify the primary keyword cluster, set the CTA tier, and decide which platforms should be prioritised. Summarise your plan and list which context will handle each major task.”

Use desktop prompts to make decisions that set the direction for the rest of the run.
