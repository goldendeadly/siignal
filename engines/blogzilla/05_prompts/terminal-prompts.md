# Terminal Prompts

Terminal prompts are used for bulk processing tasks—summarising outputs, generating trackers, or producing reports. They are executed in a command‑line or script environment that can read multiple files, perform operations and write results.

Example prompt:

“You are operating in terminal mode. Read all files in `04_workspaces/repurposing-engine/[run-folder]/`. Create a platform output index summarising each asset (platform, file name, CTA, status). Save the index as `platform-output-index-[topic]-v1.md` in the run folder. Then generate a summary table for the tracker.”

Use terminal prompts when processing a batch of files or generating aggregated reports that feed into the trackers.
