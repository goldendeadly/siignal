# Analytics & Memory Feedback Template

This template captures performance metrics and reflections for each run. Suggested fields:

| Metric | Data Source | Description |
| --- | --- | --- |
| CTR (%) | Platform analytics | Click‑through rate |
| Engagement Rate (%) | Platform analytics | Combined likes, comments, shares |
| Impressions | Analytics API | Number of views or reach |
| Open Rate (%) | Email platform | Percentage of opens for newsletters |
| Keyword Rank Movement | Search Console | Change in search position |
| Backlinks Added | Manual entry | Number of new backlinks gained |
| Performance Score | Formula | Weighted composite score across metrics |
| Top Performing Platform | Lookup | Platform with highest resonance score |
| Notes / Lessons Learned | Long text | Insights, surprises and recommendations |
| Date of Reflection | Date | When the reflection was recorded |

Use this template to build weekly or monthly review views in your tracker. Automate metrics ingestion where possible and update the reflection field manually after analysis.
