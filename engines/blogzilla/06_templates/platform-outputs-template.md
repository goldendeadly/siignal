# Platform Outputs Tracker Template

Use this template to track each content piece produced for different platforms. Recommended fields:

| Field | Type/Format | Description |
| --- | --- | --- |
| Platform | Single select | e.g. X/Twitter, LinkedIn, Instagram, YouTube, Quora, Facebook, Pinterest, Medium, Substack, Blogger, Dev.to |
| Output Type | Single select | e.g. Thread, Post, Carousel, Script, Article, Pin, Digest |
| Headline/Hook | Text | Attention‑grabbing line or title |
| File Name/Link | URL or text | Location of the draft or published file |
| Hashtags/Keywords | Text | Relevant hashtags or keywords |
| CTA | Text | Call to action used |
| Publish Date | Date | Planned or actual publish date |
| Engagement (%) | Number | Engagement rate or metric for the output |
| CTR (%) | Number | Click‑through rate if applicable |
| Resonance Score (0–10) | Number | Internal rating of how well the content performed |

Create a row for each asset and link it back to the Content Master Tracker via a relational field. Use this to track performance and status across platforms.
