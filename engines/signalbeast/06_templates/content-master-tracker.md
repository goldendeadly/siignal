# Content Master Tracker Template

This template outlines the fields to record for each run in the **Content Master Tracker**.  Use it as a guide for building your spreadsheet or Airtable base.  Avoid long sentences; use short descriptions for each field.

## Fields

- **Run ID:** Unique identifier for the run (e.g. `run-2026-03-13-adaptive-prompting`).
- **Post Title:** SEO‑optimised blog title.
- **Slug / URL:** Canonical blog URL.
- **Publish Date:** Actual or planned publication date.
- **Resonance Mode:** Selected tone (Authority, Emotional, Technical, Viral, Educational, Hybrid).
- **Core Intent:** Primary objective (SEO Dominance, Lead Generation, etc.).
- **Primary Keyword:** Core keyword focus.
- **LSI / Secondary Keywords:** Supporting semantic keywords (comma‑separated list).
- **Search Intent:** Informational, Educational, Transactional, Philosophical, etc.
- **Summary:** 2–3 sentence overview of the source (copy from `summary.md`).
- **Meta Description:** 150–160 character snippet (copy from `meta-description.md`).
- **Key Highlights:** List of 5–10 standalone insights with tags.
- **CTA Tier Used:** Soft, Engagement or Conversion.
- **Canonical Statement:** The exact canonical line used in outputs.
- **Status:** Idea, Draft, Review, Scheduled, Published, Distributed, Archived.
- **Assigned To:** Name of collaborator responsible for this run.
