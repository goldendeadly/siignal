# Analytics & Memory Feedback Template

Use this template to record and reflect on performance data after content has been published.  Keep entries concise and structured for quick reference.

## Fields

- **Metric:** Name of the metric (CTR %, Engagement Rate %, Impressions, Opens %, Keyword Rank, Backlinks Added, etc.).
- **Data Source:** Platform analytics or tool used (Search Console, platform dashboard, CRM, etc.).
- **Value:** Numeric value of the metric.
- **Performance Score:** Calculated score based on defined weighting (e.g. `(CTR×0.4)+(Engagement×0.3)+(Opens×0.3)`).
- **Top Performing Platform:** Platform with the highest resonance or performance score.
- **Notes / Lessons Learned:** Short reflections on what worked and what didn’t.
- **Date of Reflection:** When the reflection was recorded.

### Suggested Views / Automations

- **Dashboard View:** Filter to show only published and distributed outputs; sort by performance score.
- **Weekly Reflection View:** Limit to the last 7 days of metrics; focus on memory feedback.
- **Automation:** When a new record is added, trigger a notification or next step (e.g. amplify top‑performing pieces or schedule a /recur100 run).
