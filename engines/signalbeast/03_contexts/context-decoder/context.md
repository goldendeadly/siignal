# Context Decoder

## Purpose

To analyse the source content and produce a semantic map that will drive all downstream generation. The decoder identifies tone, audience intent, conversion goals, and constructs an SEO supercluster grid (primary keyword, LSI keywords, long‑tail variants, related questions, backlink concepts). It also produces the canonical statement linking back to the original blog.

## Read Before Starting

- Run brief for the current run.
- Source blog or idea text.
- Any existing research or examples in `01_source-material/`.

## Ignore

- Draft outputs from later stages.
- Tracking or expansion files (handled in later phases).

## Expected Outputs

- Tone classification (e.g. visionary, conversational educator, mythic strategic, etc.).
- Audience intent and emotional frequency profile.
- Conversion goal (SEO, lead gen, thought leadership, audience growth).
- Primary keyword, 3–5 LSI keywords, long‑tail variants and related questions.
- SEO supercluster grid summarising keyword relationships and questions.
- Canonical statement referencing the original blog.
