## Recursion Logic

### Feedback Loop Integration
After initial deployment, the Signal Storm evolves based on:

1. **Engagement Signals** — Which posts got the most likes/shares/comments?
2. **Click-Through Data** — Which hooks drove the most traffic?
3. **Bounce Rate** — Which landing page elements caused drop-off?
4. **Comment Sentiment** — What language did the audience use in responses?

### Evolution Rules
- **High-performing hooks** → Amplify and create 3 more variants
- **Low-performing formats** → Swap style (e.g., authority → empathy)
- **Audience language** → Inject their exact words into next-gen copy
- **Platform winners** → Double down on that channel's output volume
