# Signal Expansion Workflow

This workflow guides you through expanding the content signal created in earlier phases.  The goal is to derive new content opportunities and growth strategies from the data and outputs collected so far.

## Steps

1. **Review analytics and performance**
   - Read the **Analytics & Memory Feedback** document to understand which platforms, topics and content types performed best.
   - Note which keywords or highlights generated the most engagement and clicks.

2. **Brainstorm new topics**
   - Using the high‑performing keywords and latent topic gaps identified in the context analysis, generate five potential blog topics.
   - Ensure each topic is aligned with the niche and resonance mode of the current run.
   - Write a one‑sentence description of what each topic would cover.

3. **Identify collaborators and backlink targets**
   - Research three potential collaborators—bloggers, YouTubers, podcasters, Twitter accounts or communities—who produce complementary content.
   - For each collaborator, note why they would be a good fit (audience overlap, authority level, potential for mutual benefit).
   - Identify websites or publications where a guest post could create high‑quality backlinks.

4. **Propose a lead magnet**
   - Based on the source material and performance signals, suggest one valuable resource that could be offered as a download (e.g. an infographic, checklist, mini‑guide or template).
   - Outline its key elements and explain how it ties back to the main topic.

5. **Recommend a system improvement**
   - Examine the workflow and note any bottlenecks, manual tasks or inefficiencies from the current run.
   - Suggest one automation, tool integration or process change to streamline future runs (e.g. using Zapier to auto‑update trackers, scheduling posts with Buffer, automating lead magnet delivery via email CRM).

6. **Save the expansion output**
   - Write the results into a Markdown file named `expansion-recommendations-v1.md` in `04_workspaces/signal-expansion/[run-folder]/`.
   - Organise sections clearly: **New Topics**, **Collaborators & Backlinks**, **Lead Magnet Idea**, **System Improvement**.

7. **Hand off to Automation Suggestions**
   - Once the expansion ideas are documented, proceed to the Automation Suggestions phase to design the implementation of the recommended improvement.

## Outputs

- `expansion-recommendations-v1.md` containing new topics, collaboration/backlink targets, a lead magnet idea, and a process improvement suggestion.
