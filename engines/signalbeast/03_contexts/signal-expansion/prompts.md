# Signal Expansion Prompts

The prompts in this file support the ideation and analysis tasks required in the **Signal Expansion** phase.

## Thinking Task Prompt

> **Task:** Generate new content and growth opportunities based on performance data.
>
> **Read:**
> - The **Analytics & Memory Feedback** document from the Tracking & Canon phase.
> - The original context analysis for keyword and topic references.
> - All platform outputs from the current run.
>
> **Steps:**
> - Identify which topics, keywords and formats performed best.
> - Brainstorm five new blog or content ideas that build on these successes.
> - Consider collaborations or backlink opportunities and why they align with the topic.
> - Propose a lead magnet concept that would provide value to the audience.
> - Note one improvement or automation that would enhance future runs.
>
> **Output:** A short list of new topics, collaborators/backlink targets, a lead magnet idea and a suggested system improvement.

## File Editing Task Prompt

> **Task:** Document the expansion recommendations.
>
> **Read:**
> - The output from the thinking task.
>
> **Steps:**
> - Create a Markdown file `expansion-recommendations-v1.md` in the current run’s signal-expansion workspace.
> - Organise the document into sections: New Topics, Collaborators & Backlink Targets, Lead Magnet Idea, System Improvement.
> - Write concise bullet points or short paragraphs for each section.
> - Ensure the document is clear and actionable for the Automation Suggestions phase.
>
> **Output:** A well‑structured expansion recommendations file saved in the run workspace.

## Folder Processing Task Prompt

> **Task:** Aggregate multiple expansion recommendation drafts.
>
> **Read:**
> - All `expansion-recommendations-v1.md` files in the current run folder (if multiple team members contributed ideas).
>
> **Steps:**
> - Combine the new topics lists, removing duplicates and grouping similar ideas.
> - Collate all collaborator/backlink suggestions and prioritise them based on alignment and feasibility.
> - Consolidate lead magnet ideas; if multiple formats are suggested, choose the one with the broadest appeal.
> - Merge system improvement suggestions and select the most impactful one for the next run.
>
> **Output:** A final `expansion-recommendations-v2.md` file representing the combined best ideas.
