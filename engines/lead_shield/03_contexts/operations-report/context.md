# Context: Operations Report

## Role
Weekly aggregation of daily digests into a comprehensive portfolio health report. This provides strategic-level visibility into operational trends, maintenance backlogs, and communication patterns across all managed properties.

## Scope
- Aggregates 5–7 daily digests into a weekly summary
- Identifies trends (increasing call volume, recurring issues, problem properties)
- Calculates portfolio-wide KPIs
- Provides recommendations for operational improvements

## Inputs
- All daily digests from the reporting period (Monday–Friday or Monday–Sunday)
- Historical weekly reports (for trend comparison)
- Property portfolio metadata (total units, occupancy rates)

## Outputs
- Weekly Operations Report containing:
  - Executive summary (3-sentence overview)
  - Volume trends (week-over-week comparison)
  - Response time performance (vs. SLA targets)
  - Maintenance backlog status
  - Top 5 properties by call volume
  - Owner communication log
  - Recommendations (3–5 actionable items)

## Success Criteria
- Must be delivered by Monday 9:00 AM
- Must cover all managed properties
- Must include week-over-week trend data
- Must provide actionable recommendations (not just data)
