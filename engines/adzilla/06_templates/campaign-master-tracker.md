# Campaign Master Tracker Template

This template provides the structure for a campaign‑level tracker.  Use it to record overarching details about each run and link to individual ad performance records.

## Fields

- **Run ID:** A unique identifier for the campaign (e.g. `run-2026-03-13-electric-scooter`).
- **Campaign Name / Product:** The product or service being promoted.
- **Resonance Mode:** Authority, Emotional, Technical, Viral, Educational, Hybrid.
- **Marketing Objective:** Awareness, Lead Generation, Sales, Retargeting, etc.
- **Budget (Total):** Planned total spend for the campaign.
- **Budget (Daily):** Daily budget if applicable.
- **Start Date:** Campaign start date.
- **End Date:** Campaign end date or planned end date.
- **Platforms:** List of platforms used.
- **KPIs:** Key performance indicators being tracked (CPC, CTR, Conversions, ROAS, etc.).
- **Status:** Idea, Planning, Active, Paused, Completed, Archived.
- **Notes:** Additional notes or context for the campaign.
