# Ad Performance Tracker Template

Track every ad’s performance data in this table.  Each row represents a single ad variation on a single platform.

## Fields

- **Run ID:** Link to the campaign master tracker.
- **Ad ID / File Name:** Name of the ad file (e.g. `electric-scooter-facebook-ad-final-v1.md`).
- **Platform:** Facebook, Instagram, Google Ads, LinkedIn, YouTube, TikTok, etc.
- **Headline / Hook:** The main attention‑grabbing line.
- **CTA:** Call to action used.
- **Target Audience Segment:** Which audience segment this ad targets.
- **Impressions:** Number of times the ad was shown.
- **Clicks:** Number of clicks recorded.
- **CTR (%):** Click‑through rate.
- **Conversions / Actions:** Number of desired actions (purchases, signups, etc.).
- **Cost per Click (CPC):** Advertising spend divided by clicks.
- **Cost per Acquisition (CPA):** Advertising spend divided by conversions.
- **Return on Ad Spend (ROAS):** Revenue generated divided by spend.
- **Performance Score:** Weighted score based on chosen formula.
- **Status:** Draft, Active, Paused, Completed, Archived.
- **Notes:** Additional observations or context.
