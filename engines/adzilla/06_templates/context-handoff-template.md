# Context Handoff Template

This template documents the transfer of work from one AdZilla context to another.  Use it to ensure a smooth transition and avoid information loss.

## From:

Specify the originating context (e.g. Offer Intake, Market Analysis).

## To:

Specify the destination context (e.g. Creative Generation, Ad Assembly).

## Summary of Completed Work

Describe what has been completed in the current context (e.g. offer details captured, audience analysis completed, messaging assets created).

## Files to Read Next

List all files that the next context must read (e.g. `offer-details.md`, `market-analysis-v1.md`, `value-propositions.md`, ad drafts).

## Notes & Considerations

Highlight any constraints, assumptions, questions or dependencies that the next context should know.

## Next Steps

Provide a short checklist of tasks for the next context based on its workflow (e.g. write ads, plan targeting, log metrics).
