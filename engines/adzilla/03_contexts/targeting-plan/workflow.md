# Targeting Plan Workflow

This workflow defines the process for creating a detailed targeting plan tailored to each advertising platform.

## Steps

1. **Review run brief and audience persona**
   - Confirm the overall budget, campaign duration and selected platforms.
   - Revisit the audience insights from `market-analysis-v1.md` to align targeting with the persona’s demographics and interests.

2. **Define audience segments**
   - Create segments based on demographics (age, gender, location), interests (hobbies, topics, brands) and behaviours (purchase history, device usage).
   - Consider custom audiences (e.g. website visitors, email subscribers) and lookalike audiences for platforms that support them.
   - Document each segment’s name, criteria and rationale.

3. **Choose geographic targeting**
   - Specify geographic areas for each segment (country, region, city or radius around a specific location).
   - Note any exclusions (e.g. regions where the product is unavailable).

4. **Allocate budgets**
   - Determine how to distribute the overall budget across platforms and segments.
   - Choose between daily and lifetime budgets; set maximum bids or cost caps if applicable.
   - Record the budget in currency units and track whether budgets are flexible or fixed.

5. **Set schedules**
   - Define start and end dates for each campaign or ad set.
   - Implement dayparting if certain times or days are better for reaching the audience.
   - Note any events or seasonal considerations that could affect scheduling.

6. **Select optimization goals**
   - Choose a primary optimization goal for each platform (e.g. clicks for awareness, conversions for sales).
   - Decide on bidding strategies (manual vs. automated, cost‑per‑click vs. cost‑per‑action).

7. **Document the plan**
   - Create `targeting-plan-v1.md` with sections for each platform, detailing audience segments, geographic targeting, budgets, schedules and optimization goals.
   - Use clear headings and bullet points.

8. **Hand off to Performance Tracking**
   - Once the targeting plan is approved, proceed to the Performance Tracking context to execute the campaign and monitor results.

## Outputs

- A comprehensive targeting plan file ready for campaign execution.
