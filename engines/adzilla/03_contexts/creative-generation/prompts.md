# Creative Generation Prompts

These prompts help you think through and produce the core messaging assets in the Creative Generation phase.

## Thinking Task Prompt

> **Task:** Translate audience insights into compelling messages.
>
> **Read:**
> - `market-analysis-v1.md` for audience and competitor insights.
> - `offer-details.md` for product features and benefits.
>
> **Steps:**
> - Identify the top two or three reasons someone should choose this offer over alternatives.
> - List the features that deliver tangible benefits and convert them into customer‑centric statements.
> - Develop hooks that will grab attention in the first few seconds or words.
> - Brainstorm taglines that encapsulate the offer’s promise memorably.
>
> **Output:** A draft set of value propositions, benefit bullets, hooks and taglines to be refined.

## File Editing Task Prompt

> **Task:** Finalise and save the messaging assets.
>
> **Read:**
> - Draft messaging from the thinking task.
>
> **Steps:**
> - Write `value-propositions.md` listing the final value propositions.
> - Write `benefits.md` listing the final benefit bullets.
> - Write `hooks-taglines.md` with separate sections for hooks and taglines.
> - Save these files in `04_workspaces/creative-generation/[run-folder]/`.
>
> **Output:** Completed messaging files ready for the Ad Assembly phase.

## Folder Processing Task Prompt

> **Task:** Combine multiple messaging drafts.
>
> **Read:**
> - All `value-propositions.md`, `benefits.md` and `hooks-taglines.md` drafts if multiple writers contributed.
>
> **Steps:**
> - Merge all value propositions into one list, removing duplicates.
> - Consolidate benefit bullets, grouping similar benefits and ensuring a variety of angles (emotional, functional, economic).
> - Combine hooks and taglines into their respective lists, selecting the best examples and discarding weaker ones.
> - Save the consolidated versions with the next version number (e.g. `value-propositions-v2.md`).

> **Output:** Unified messaging files representing the best ideas from all contributors.
