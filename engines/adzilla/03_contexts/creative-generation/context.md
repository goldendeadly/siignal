# Creative Generation Context

## Purpose

The **Creative Generation** context transforms the market insights into compelling messaging assets.  Here you craft value propositions, benefit bullets, hooks and taglines that will become the foundation of your advertising copy across platforms.  This phase demands creativity, clarity and alignment with the defined resonance mode.

## Read First

- The market analysis document (`market-analysis-v1.md`).
- The offer details file (`offer-details.md`).
- Run brief for reference on resonance mode and objectives.

## Use This Context For

- Summarising the offer’s unique selling proposition into concise **value propositions** (one or two sentences each).
- Crafting **benefit bullets** that highlight specific features and advantages, phrased in audience‑centred language.
- Generating **hooks** designed to capture attention in the first few words of an ad (questions, surprising facts, bold statements).
- Writing **taglines** or slogans that encapsulate the brand promise in a memorable way.
- Tagging each message with its type (value proposition, benefit, hook, tagline) for easy reference.

## Ignore

- Platform‑specific formatting (handled in Ad Assembly).
- Budget or targeting considerations (handled later).

## Expected Outputs

- `value-propositions.md` – A list of 2–3 succinct value propositions.
- `benefits.md` – A list of 5–10 benefit bullets highlighting features and outcomes.
- `hooks-taglines.md` – A combined file containing hooks and taglines separated by section, each clearly labelled.
