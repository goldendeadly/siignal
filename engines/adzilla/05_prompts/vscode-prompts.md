# VS Code Prompts

These prompts are used in an IDE such as VS Code to generate or edit files within a specific AdZilla context.

## Context Task Prompt

> **Objective:** Produce deliverables for a single context.
>
> **Read:**
> - The `context.md` and `workflow.md` for the current context (located in `03_contexts/[context-name]/`).
> - Relevant input files in `04_workspaces/[context-name]/[run-folder]/`.
>
> **Tasks:**
> - Follow the workflow steps to create or update the required files.
> - Use the proper naming conventions as defined in `00_router/naming-conventions.md`.
> - Save the outputs in the run’s workspace folder.
>
> **Output:** Completed files for the context (e.g. `offer-details.md`, `market-analysis-v1.md`, ad files, trackers, optimisation reports).

## Ad Drafting Prompt

> **Objective:** Write or revise ad copy for a particular platform.
>
> **Read:**
> - `value-propositions.md`, `benefits.md`, `hooks-taglines.md` from creative generation.
> - The plan from the thinking task in Ad Assembly.
> - Platform guidelines (e.g. character limits, best practices).
>
> **Tasks:**
> - Compose headlines, body copy, CTAs and creative notes tailored to the platform.
> - Incorporate resonance mode and key messaging points.
> - Include a canonical brand line.
> - Save the draft using the correct filename in the ad‑assembly workspace.
>
> **Output:** A polished ad file ready for review.
