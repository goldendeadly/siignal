# Strategy Modeling Context

## Purpose

Translate the validated run brief into a comprehensive **strategy model** that will guide all downstream content creation.  This context synthesises avatar psychology, offer architecture, tone guidelines, desired outcome mapping and SEO keyword clusters.

## Read First

* The validated run brief in the current run folder (`04_workspaces/input-intake/`).
* Templates in `06_templates/`:
  * `avatar-map-template.md`
  * `offer-architecture-template.md`
  * `seo-intent-template.md`

## Use This Context For

* Building the avatar map (demographics, psychographics, pains and desires).
* Describing the offer’s features, benefits, promises, pricing and guarantees.
* Calibrating the conscious frequency / tone across all content.
* Mapping the primary outcome or transformation the customer seeks.
* Creating the SEO intent map (primary keyword, LSI terms, long‑tail queries, search intent).

## Ignore

* Do not generate blog titles, posts or social content here.
* Do not write ad copy or landing pages.

## Expected Outputs

* A **strategy model** markdown file saved in `04_workspaces/strategy-modeling/` following the naming convention `[run-slug]-strategy-model-core-v1.md`.
