# Strategy Modeling Prompts

Use these prompts to guide your thinking, file creation and folder processing in the strategy‑modeling context.

## Thinking Prompt

> **Question:** Who is the ideal customer (avatar) for this offer, what pains do they feel, what desires motivate them, and how will the product transform them?
>
> **Action:** Reflect on the run brief and any research notes.  Build a detailed avatar map and connect the dots between the avatar’s pains, the offer’s benefits and the desired transformation.  Decide how the chosen tone should influence messaging.

## File Editing Prompt

> **Task:** Write the strategy model for this run.
>
> **Steps:**
> 1. Create sections for Avatar, Offer, Tone, Outcome and SEO Intent.
> 2. Use the relevant templates in `06_templates/` to capture details in each section.
> 3. Summarise findings and note any assumptions or questions.
> 4. Save as `[run-slug]-strategy-model-core-v1.md` in `04_workspaces/strategy-modeling/`.

## Folder Processing Prompt

> **Task:** Organise the inputs and outputs for the strategy phase.
>
> **Steps:**
> 1. Ensure the validated run brief is present in the run folder.
> 2. Place any research notes or competitor examples in the run folder for reference.
> 3. After creating the strategy model, copy it to `08_outputs/strategy-models/` (but keep the working copy in the workspace until packaging).
