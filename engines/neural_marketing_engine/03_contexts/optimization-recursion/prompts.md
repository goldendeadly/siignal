# Optimisation & Recursion Prompts

## Thinking Prompt

> **Question:** What insights can be drawn from performance data and feedback, and how can the existing assets be improved to better resonate with the avatar and achieve the desired outcome?
>
> **Action:** Analyse metrics and comments.  Identify patterns (e.g. low CTR on certain headlines or high drop‑off points on the landing page).  Decide whether to adjust messaging, tone, length, keywords or CTA placement.  Ensure changes remain aligned with the strategy model.

## File Editing Prompt

> **Task:** Revise the underperforming assets.
>
> **Steps:**
> 1. Open the original file in its workspace.
> 2. Apply targeted edits (e.g. rewrite headlines, reorganise paragraphs, adjust CTAs, refine keyword usage).
> 3. Save the revised file with an incremented version number (e.g. `v2`).
> 4. Document what changed and why in a `feedback-summary.md` file.

## Folder Processing Prompt

> **Task:** Organise revisions and update the run folder.
>
> **Steps:**
> 1. Ensure all revised files are saved in their respective workspaces with correct version numbers.
> 2. Copy revised files to `08_outputs/recursion-revisions/` while retaining earlier versions for comparison.
> 3. Save the feedback summary in `04_workspaces/optimization-recursion/`.
