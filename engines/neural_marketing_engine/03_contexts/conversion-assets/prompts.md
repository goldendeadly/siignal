# Conversion Assets Prompts

## Thinking Prompt

> **Question:** What messaging will compel the avatar to take action across different advertising channels, and how should the tone shift to match the context?
>
> **Action:** Consider the avatar’s primary pain and desired outcome.  For each channel, decide whether to lead with emotion, authority, urgency or curiosity.  Brainstorm headlines and CTAs that align with the offer’s promise and the platform’s norms.

## File Editing Prompt

> **Task:** Write the ad copy sets and landing page draft.
>
> **Steps:**
> 1. Create headings for each advertising channel and list three variations under each.
> 2. Follow the channel’s character limits and guidelines.  Always include a clear CTA.
> 3. Outline and draft the landing page using the conversion assets template as a guide.
> 4. Save the ad copy as `[run-slug]-ad-copy-conversion-v1.md` and the landing page as `[run-slug]-landing-page-conversion-v1.md` in the conversion assets workspace.

## Folder Processing Prompt

> **Task:** Organise conversion assets for packaging.
>
> **Steps:**
> 1. Verify that both the ad copy file and landing page file exist in `04_workspaces/conversion-assets/`.
> 2. Copy them to `08_outputs/conversion-assets/` when ready for final packaging.
> 3. If the optimisation recursion context will run, retain the original versions for comparison with revised versions.
