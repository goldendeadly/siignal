# Signal Expansion Prompts

## Thinking Prompt

> **Question:** Which insights from the blog will resonate most strongly on each platform, and how should the tone shift to fit each channel?
>
> **Action:** Identify the core ideas that will inspire, educate or provoke the avatar.  Decide how to adapt the conscious frequency (e.g. more conversational on Instagram, more authoritative on LinkedIn) while preserving the core message.  Plan the narrative arc for threads and videos.

## File Editing Prompt

> **Task:** Draft the multi‑platform social signal file.
>
> **Steps:**
> 1. Create section headings for each platform.
> 2. Under each heading, write the required number of posts, tweets, carousel slides or scripts.
> 3. Include hooks, insights, CTAs, hashtags and formatting notes.  Use bullet points or numbered lists for clarity.
> 4. Save the compiled file as `[run-slug]-social-signal-expansion-v1.md` in the signal expansion workspace.

## Folder Processing Prompt

> **Task:** Organise social signal files and prepare for the next stage.
>
> **Steps:**
> 1. Ensure the social signal file is saved in `04_workspaces/signal-expansion/`.
> 2. If individual platform sub‑files are created (optional), ensure their names follow the naming conventions (e.g. `[run-slug]-twitter-thread-v1.md`).
> 3. Copy final versions to `08_outputs/social-signal/` when entering the packaging context.
