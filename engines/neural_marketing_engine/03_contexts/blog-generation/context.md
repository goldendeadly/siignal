# Blog Generation Context

## Purpose

Create the primary content asset: a set of potential blog titles and a long‑form blog post grounded in the strategy model.  This context focuses on ideation and long‑form writing, not on distribution or optimisation.

## Read First

* The **strategy model** from the previous phase (in `04_workspaces/strategy-modeling/`).
* The run brief to refresh the tone, outcome and SEO intent.

## Use This Context For

* Generating seven compelling blog title options.
* Selecting (or ranking) the best title for the blog post.
* Writing a full blog post (1,500–2,500 words) aligned with the avatar’s pain points, the offer’s promise and the selected conscious frequency.

## Ignore

* Do not generate social media posts or ad copy here.
* Do not perform keyword research (already defined in the strategy model).

## Expected Outputs

* A markdown file containing seven blog titles, saved as `[run-slug]-blog-titles-generation-v1.md`.
* A markdown file containing the long‑form blog post, saved as `[run-slug]-blog-post-generation-v1.md`.
* Both files should reside in `04_workspaces/blog-generation/` until ready for packaging.
