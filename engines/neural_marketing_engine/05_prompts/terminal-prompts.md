# Terminal Prompts – Neural Marketing Engine

Terminal prompts are used for batch operations, summaries and file processing tasks that do not involve writing content.  They are helpful for indexing, moving files and producing consolidated views.

## How to Use

Use a terminal prompt when you need to:

* Batch‑process a set of files (e.g. combine multiple social posts into one document).
* Summarise the contents of a folder (e.g. create a table of all outputs with links).
* Perform housekeeping tasks such as copying final versions into the outputs folder or archiving old runs.

## Template

```
**Task:** Describe the batch operation (e.g. generate the output index).

**Source Folder:** Specify the path to the folder you will read from (e.g. `04_workspaces/blog-generation/`).

**Destination Folder:** Specify where the resulting file should be saved (e.g. `08_outputs/final-delivery/`).

**Instructions:** Provide step‑by‑step directions for the file operations (e.g. list all markdown files, extract titles, build a table, save the result).
```

Terminal prompts keep file operations deterministic and reproducible.
