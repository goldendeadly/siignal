# VS Code Prompts – Neural Marketing Engine

VS Code prompts are used when generating or editing files within a context.  They ensure that the AI reads the correct inputs, follows the workflow instructions and produces the expected outputs.

## How to Use

Use a VS Code prompt when:

* Writing the strategy model, blog post, social signal, ad copy, landing page or revised assets.
* Filling out templates such as avatar maps, offer architecture or SEO intent maps.

## Template

```
**Context:** Specify the current context (e.g. blog-generation, signal-expansion).

**Files to Read:** List the files you need to open (strategy model, run brief, previous versions).

**Task:** Describe what you need to create or edit (e.g. write a long‑form blog post).

**Instructions:** Summarise the relevant steps from the context’s workflow.

**Output:** Specify the file name and location where the new or updated file should be saved.
```

By including the context name and file paths, you ensure that the AI stays within the correct workspace and follows the established naming conventions.
