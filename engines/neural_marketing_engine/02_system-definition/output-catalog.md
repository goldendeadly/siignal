# Output Catalog – Neural Marketing Engine v1

This catalog lists the assets produced by the engine and provides a brief description of each.  Use it as a checklist when packaging deliverables.

| Output | Description | Location |
|------|-------------|----------|
| **Run Brief** | The initial intake document capturing niche, offer, avatar, tone, outcome and SEO intent.  Acts as the source of truth for the run. | `07_inputs/run-briefs/` and `04_workspaces/input-intake/` |
| **Strategy Model** | A comprehensive model combining avatar map, offer architecture, tone guidelines, outcome mapping and SEO cluster.  Guides all downstream content creation. | `08_outputs/strategy-models/` |
| **Blog Title List** | Seven suggested blog headlines aligned to the strategy model.  Helps select the most compelling title before writing. | `08_outputs/blog-assets/` |
| **Long‑Form Blog Post** | A 1,500–2,500 word article built on the chosen title and strategy model.  Includes an introduction, body sections, conclusion, and soft CTA. | `08_outputs/blog-assets/` |
| **Social Signal** | A multi‑platform content set derived from the blog.  Includes threads (Twitter/X), carousels (LinkedIn/Instagram), short video scripts (YouTube Shorts/Reels/TikTok) and captions. | `08_outputs/social-signal/` |
| **Ad Copy Set** | A collection of ad headlines and bodies for various channels (search, social, display).  Incorporates different CTA tiers and resonates with the avatar. | `08_outputs/conversion-assets/` |
| **Landing Page Draft** | A page structure containing headline, problem, promise, benefits, proof points, CTA stack and form instructions.  Written to convert the avatar based on the offer. | `08_outputs/conversion-assets/` |
| **Feedback Revisions** | Revised versions of any of the above assets produced during optimisation.  Each revision increments the version number. | `08_outputs/recursion-revisions/` |
| **Final Delivery Bundle** | A compiled package containing final versions of all assets and an output index summarising them.  Ready for hand‑off to clients or stakeholders. | `08_outputs/final-delivery/` |
| **Output Index** | A markdown file listing all deliverables, their file names, versions and descriptions.  Helps recipients locate each asset quickly. | `08_outputs/final-delivery/` |

When packaging a run, verify that each item above exists and is placed in the correct folder.  Use the naming conventions from `00_router/naming-conventions.md` to ensure consistency.
