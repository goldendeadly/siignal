# Terminology – Neural Marketing Engine v1

This glossary defines key terms used within the Neural Marketing Engine.  Refer to it whenever a concept is unclear.

| Term | Definition |
|-----|------------|
| **Avatar** | A composite profile representing the ideal customer.  Includes demographic details (age, gender, location), psychographic traits (values, interests, beliefs), pain points, desires and objections. |
| **Conscious Frequency** | A tonal calibration specifying the emotional resonance of the copy.  Examples include Authority (confident, data‑backed), Emotional (story‑driven, sincere), Technical (analytical, process‑oriented), Viral (bold, provocative) and Educational (clear, instructive). |
| **Primary Outcome / Transformation** | The main result the customer seeks by engaging with the offer.  For example, “lower energy bills”, “lose weight”, or “improve workplace productivity”. |
| **SEO Intent** | The search intent guiding keyword selection.  Categories include Informational (learn about a topic), Transactional (ready to buy), Navigational (looking for a specific site), and Comparison (evaluating options). |
| **Strategy Model** | A document produced during the strategy‑modelling phase.  It synthesises avatar insights, offer framing, tone guidelines, outcome mapping and an SEO keyword cluster.  All subsequent outputs rely on this model. |
| **Social Signal** | The collection of platform‑tailored posts derived from the blog (threads, carousels, captions, scripts).  Its purpose is to broadcast the blog’s insights across different channels while maintaining consistency. |
| **Recursion** | A process of iterative improvement where performance feedback is analysed and used to revise the blog post, social signal, ads and landing page.  Each iteration increments the version number. |
| **Run Slug** | A hyphen‑separated identifier combining the niche and offer (e.g. `roofing-leadgen`).  Used in file and folder names. |
